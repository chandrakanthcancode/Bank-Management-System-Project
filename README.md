# Bank Management System (Core Java) 
This is a simple Java console-based Bank Management System using Core Java features:
  1. Create account
  2.  Deposit money 
  3.  Withdraw money
  4.  Check balance
  5.   View all accounts

 ## How to Run 
 
 1. Save the file as Main.java
 2. Compile:javac Main.java
 3. Run: java Main
